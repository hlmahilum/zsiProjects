DASHBOARD

Module Name: Dashboard

Description: After every successful login, the user will be redirected to the Dashboard Page. There are three Types of Dashboard used in this system: Inventory Dashboard, Procurement Dashboard, and Aircraft Dashboard. Some Accounts were given access on all dashboards.



INVENTORY DASHBOARD

Module Name: Inventory Dashboard



    1. Logo

    2. Account Name, User Manual, Logout

    3. Menu Bars

    4. Dashboard Title and Warehouse Location Selection Box

    5. Show Option Selection-box

    6. Show Option Filter and Clear Button

    7. Data Tab Panels

    8. Data Grid/Table







1. Logo

» The Logo is displayed at the top-left part of every page, and is a clickable link to Dashboard Page. It can be used as a shortcut link from any page back to the dashboard page.





2. Account Name, User Manual, Logout

» The Account Name (Role) found at the upper-right side of every page displays the Account Name with its Designated Role that was used by the user in Logging-in the system. It also houses the drop-down menus for User Manual, Change Password, and Logout.





3. Main Menu Navigation Bar

» This bar displays the headings and houses drop-down menus that allow the user to interact with the content or application in various ways. Commands are grouped under each of these menu headings according to function.





4. Dashboard Title and Warehouse Location Selection Box

» Displays the dashboard title and warehouse location. Some accounts are allowed to switch or select warehouse location.





5. Show Option Selection-box

» For a filtered data display, type or enter the desired entries on the input textbox then click the filter button.





6. Show Option Filter and Clear Buttons

» Clicking the filter button without entries on the show option input box will display all available data. The clear button will clear the entries made in the show option selection-boxes.





7. Data Tab Panels

» Inventory dashboard has five tab panels namely: Assembly, Components, Equipment, Tools, and Supply. Click on the title of the tab panel to switch from one panel to another.





8. Data Grid/Table

» Displays the records or the data of each Tab Panel.
