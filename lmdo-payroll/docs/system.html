System User

Module Name: System User



Description: Located at the rightmost of the main menu. It displays the name and the role assigned to the System User. On it's Submenu are the User Manual, the Change Password, and the Logout click-select options.









System User ► Help (User Manual)

Module Name: Help User Manual

Description: View and access Help (User Manual)

Program Flow:



» Under the logged USERNAME located at rightmost of the main menu , select to open/view the User Manual Page.











System User ► Change Password

Module Name: Change Password

Description: How to Change Password

Program Flow:



» Under the logged USERNAME located at rightmost of the main menu , select to access the Change Password Page.











System User ► Logout

Module Name: Logout

Description: How to logout of the system.

Program Flow:



» To logout of the system, select under the logged USERNAME located at rightmost of the main menu. In doing so, the user is redirected to the Landing Page and must supply Username-Password to login back of the system.
