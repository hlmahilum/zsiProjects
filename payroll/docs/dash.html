MAIN DASHBOARD

Module Name: Main Dashboard

Description: After every successful login, the user will be redirected to the Main Dashboard Page. The Main Dashboard contents and view will vary depending on the Role designated to the User.



    1.Logo

    2.Menu

    3.Account Name

    4.Logout

    5. Title with Organization Name/Warehouse Location

    6. Show Option/Filter

    7. Tab Panels

    8. Table



1. Logo

» The Logo is located at the top-leftmost of every page is a clickable link to Dashboard Page. It can be used as a shortcut link from any page back to the dashboard page.





2. Main Menu Navigation Bar

» This bar displays the headings and houses drop-down menus that allow the user to interact with the content or application in various ways. Commands are grouped under each of these menu headings according to function.





3. Account Name (Role)

» The Account Name (Role) located at the upper-right side of every page displays the Account Name with its Designated Role that was used by the user in Logging-in the system.





4. Log Out

» Used to Log-out of the system. After clicking the the Log-out Button, the user will be redirected back to the Landing Page.





5. Title with Organization Name and Warehouse Location

» Displays The Main Dashboard Title , name of the organization, and the location of the warehouse whom the logged user belongs to.





6. Show Option/Filter

» Click-select from the given dropdown list of the Show Option , then click the filter button. Clicking the filter button with a default value will display all.





7. Tab Panels

» Click on the title of the tab panel to switch from one panel to another.





8. Table

» Displays the records or the data of each Tab Panel.
